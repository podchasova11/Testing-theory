Коротко: в Python кортеж (tuple) сам по себе неизменяемый, но если внутри него хранится изменяемый объект (например, список), то этот вложенный объект можно изменять, и такие изменения отразятся внутри кортежа.

Что это значит по шагам:
- Кортеж как контейнер не поддерживает изменение своих элементов: после создания элементы кортежа “заморожены” по ссылкам.
- Но элементы кортежа могут быть изменяемыми объектами (например, список, словарь, множество). В этом случае сам кортеж хранит ссылки на эти объекты.
- Если вы изменяете сам объект, на который ссылкается элемент кортежа (например, изменяете содержимое списка внутри кортежа), то вид кортежа снаружи может казаться изменённым, потому что внутри кортежа лежит тот же самый изменяемый объект.
- Если же вы попробуете заменить элемент кортежа другим объектом (например, присвоить новый список по индексу), то получите ошибку TypeError, потому что элементы кортежа недоступны для присваивания.

Примеры:

- Изменение содержимого вложенного списка (внутри кортежа):
```python
t = (1, [2, 3], "a")
print(t)  # (1, [2, 3], 'a')

t[1].append(4)  # изменяем вложенный список
print(t)  # (1, [2, 3, 4], 'a')
```
Здесь кортеж остался тем же объектом, но внутри него изменился список.

- Замена элемента кортежа не работает:
```python
t = (1, [2, 3], "a")
t[1] = [9, 9]  # TypeError: 'tuple' object does not support item assignment
```

- Замена вложенного объекта целиком возможна только если у вас есть ссылка на этот объект (или создаёте новый кортеж):
```python
t = (1, [2, 3], "a")
lst = t[1]          # ссылка на вложенный список
lst = [9, 9]        # теперь 'lst' указывает на новый список, но кортеж не изменился
print(t)              # (1, [2, 3], 'a')

# чтобы обновить кортеж приходится создать новый кортеж:
t = (t[0], lst, t[2])
print(t)              # (1, [9, 9], 'a')
```

Итого:
- Сам кортеж неизменяемый в плане присваивания элементов.
- Вложенные изменяемые объекты могут меняться внутри кортежа без изменения самого кортежа (реальный объект остаётся тем же, но его содержимое изменяется).
- Чтобы «изменить» элементы кортежа внешне, нужно создать новый кортеж с нужными элементами.
